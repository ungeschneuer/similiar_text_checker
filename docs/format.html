format.py

Formatiert den Text so, dass Hürden zur Wert-Errechnung aus dem Weg geschafft werden. Nutzt sub aus dem Paket re

better(String) - > String

Nimmt als Input einen Text und formatiert ihn durch verschiedene Maßnahmen in eine für die Algorithmen besser verarbeitbare Version. Zuerst werden die am häufigst genutzten Abkürzungen durch volle Wörter mit abbrviations() ersetzt. Dann werden durch regex Befehle Datumausdrücke so formatiert, dass sie keine Punkte mehr haben. Ebenso werden Punkte aus Sätzen mit einem Zeichen entfernt, da dies höchst wahrscheinlich ebenfalls Abkürzungen sind. Ausgabe ist der formatierte Text.

abbreviations(String) -> String

Enthält ein dicitionary mit 200 der häufigst benutzen Abkürzungen in der deutschen Sprache. Der Input Text wird nach den Keys, also den Abkürzungen, durchsucht und, wenn eine gefunden wurde, mit dem Value, also den vollen Worten ersetzt. Ausgabe ist der neu formatierte Text

##d ialog(String) -> String Enthält eine Liste mit Formatierungen von Satzenenden, die typisch für einen Dialog sind und formatiert sie so, dass sie wie reguläre Sätze wirken. Ausgabe ist der neu formatierte Text.

directspeech(String) -> String

Enthält eine Liste von Formatierungen, die typisch für die direkte Rede sind und formatiert sie so, dass sie für die Algorithmen als Teil des umschließenden Satzes erkannt werden. Ausgabe ist der neu formatierte Text.

simplepunctuation(String) -> String

Formatiert Sätze, dass nur noch Satzzeichen übrig bleiben, die Satzteile trennen oder Sätze beenden. Formatiert diese weiterhin so, dass die Satztrennung erleichtert wird, indem alle Zeichen, die Satzenden signalisieren, durch einen Punkt ersetzt werden. Ausgabe ist der neu formatierte Text.

nopunctuation(String) -> String

Formatiert, auf den vorigen Funktionen aufbauend, den Text so, dass gar keine Satzzeichen mehr vorhanden sind. Nötig für das Splitting in Wörter. Ausgabe ist der neu formatierte Text.
